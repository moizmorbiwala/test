To run the provided code, you can follow these steps:

1. Install the required dependencies:

2. streamlit: You can install it by running pip install streamlit.

3. speech_recognition: You can install it by running pip install SpeechRecognition.

4. pyperclip: You can install it by running pip install pyperclip.

5. Save the code in a Python file with a .py extension, for example, transcription_app.py.

6. Open a command prompt or terminal and navigate to the directory where the Python file is saved.

7. Run the Streamlit app by executing the following command:

8. streamlit run transcription_app.py

9. After running the command, the Streamlit app will start, and a browser window or tab will open automatically.

10. In the Streamlit app UI, you will see the title "Real-Time Voice Transcription App" and a message to click the "Start Transcription" button and speak into your microphone.

11. Select the desired language from the dropdown menu ("English," "Spanish," or "French").

12. Click the "Start Transcription" button to begin the transcription process. Make sure you have a working microphone connected to your device.

13. Start speaking into your microphone, and the app will attempt to transcribe your speech in real-time. The transcription will be displayed on the Streamlit app UI.

14. If the transcription is successful, you will see the transcribed text on the UI. You can click the "Copy Transcription" button to copy the transcription to your clipboard.

15. If there are any issues with the transcription, such as an unknown value error or a request error, appropriate error messages will be displayed on the UI.

16. To stop the transcription process, click the "Stop Transcription" button.

Note: Make sure you have the required dependencies installed and a working microphone connected to your device for the speech recognition to work properly and proper internet connection.
